Directory structure.

This directory contains two sub-directories and four files.

The atAGlance sub-directory has text files that exhibit all language features used at a glance, in their respective language-specific files.

The summaries sub-directory holds text-files that give summaries of the game, markup, presentation and programming.

The index.html file is the markup file for this game application.

The script.js is the javascript programming file for this game application that is used by the index.html file for behaviour.

The style.css is the presentation file for this game application that is used to modify the markup display.
